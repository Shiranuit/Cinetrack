# Screenshots

Drop screenshots here and reference them from the root `README.md`.

Expected filenames (referenced by the README):

- `library.png` - the library / home screen
- `show-detail.png` - a series/movie detail page
- `discover.png` - discover / advanced filter
- `calendar.png` - upcoming & recently aired
- `profile.png` - profile with stats & favorites
- `mobile.png` - the app running on a phone

PNG or WebP, ideally captured at a consistent width. Web captures look best cropped to the
content; mobile captures look best as a device frame or a tall portrait screenshot.
